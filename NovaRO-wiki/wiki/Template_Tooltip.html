Format

This tooltip is split in three important parts:

  1. The tag <span> </span> - It's extremely important for the content to be inside a span tag, for the solely reason that a span is treated as inline, to not break the flow of the current content.
  2. The attribute class="custom-tooltip" - This attribute is targetting a custom class called "custom-tooltip", in charge of displaying the chat box above the message.
  3. The data attribute data-custom-tooltip="text goes in here" - The previous data attribute is used to add in custom content. The class custom tooltip targets the information inside of the " " throught the declaration content: attr(data-custom-tooltip);.
  • Bonus The class ("item-desc", "text" & "img") - Goes inside and next to class="custom-tooltip item-desc text img" and their solely purpose is to add extra styling to it. The order doesn't matter at long as they have an space in between them.

All of the above to construct the following:

Hover over me for some action

Use examples

Default

This code:

<span class="custom-tooltip" data-custom-tooltip="This is a very fulfilling text">Hover over me for some action</span>

Displays this:

Hover over me for some action

Using extra styling

This code:

<span class="custom-tooltip item-desc" data-custom-tooltip="And have a tooltip">I'm underlined</span>

Displays this:

I'm underlined

This code:

<span class="custom-tooltip item-desc text" data-custom-tooltip="In case it clips with the text">Moves the line downwards</span>

Displays this:

Moves the line downwards

This code:

<span class="custom-tooltip item-desc img text" data-custom-tooltip="I enable the use of the previous class with an image">{{Skill Icon|id=1}} Skill</span>

Displays this:

Skill

Using a skill

This code:

<span class="custom-tooltip" data-custom-tooltip="Gives extra damage">{{Skill List |id=7 |skill=Magnum Break Lv 5 |simple=yes}}</span>

Displays this:

Magnum Break Lv 5

Using an image

This code:

<div class="custom-tooltip" data-custom-tooltip="I'm providing context to the image" style="width: fit-content;"><div style="display: block;">[[File:Smytsu-paper-phone.png|thumb|left]]</div></div>

Displays this:

Styling the container

This code:

<span class="custom-tooltip" data-custom-tooltip="With a tooltip" style="color: white; background: darkred;">This is a very basic display</span>

Displays this:

This is a very basic display

Styling the container (advanced)

This code:

<span class="custom-tooltip" data-custom-tooltip="With a tooltip" style="color: white; background: darkred; padding: .3em .6em; border-radius: .25em; box-shadow: 0 1px 2px black, 0 2px 4px black;">This is a very fancy display</span>

Displays this:

This is a very fancy display
