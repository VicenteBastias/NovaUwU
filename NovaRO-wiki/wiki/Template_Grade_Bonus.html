Template:Grade Bonus

  • Language
  • Watch
  • Edit

Used to add grade bonus effects to description of items. Best used in combination with Template:Desc Item (or list of effects indented by :).

Syntax

{{Grade Bonus|D=|C=|B=|A=}}

Parameter
D (optional)
Grade D (or higher) bonus effect
C (optional)
Grade C (or higher) bonus effect
B (optional)
Grade B (or higher) bonus effect
A (optional)
Grade A (or higher) bonus effect

Usage

Use it to make standard item descriptions.

If you want a grade bonus effect to span multiple lines, indent the lines after the first with ::. (The first line can be skipped with <br>)

Examples

Example: 

{{Grade Bonus|D=ATK +10|C=ATK +20|B=ATK +40|A=ATK +80}}
[Bonus by grade]
[Grade D] ATK+ 10
[Grade C] ATK+ 20
[Grade B] ATK+ 40
[Grade A] ATK+ 80


{{Grade Bonus
|D=
::ATK +10
::MATK +10
|C=
::ATK +10
::MATK +10
}}
[Bonus by grade]
[Grade D]
ATK +10
MATK +10
[Grade C]
ATK +10
MATK +10
