This template displays who is currently selling the lowest priced version of the item.
 
=== Parameters ===
; id
: REQUIRED. Item ID, as defined by the NovaRO market. Be sure to use the market value over Divine-Pride's ID values (eg, Temporal Stat Boots).
; refine
: Optional. For weapons/armours, use this to retrieve the whosells of a specific refine level only. If unspecified, then the currently lowest priced item is shown regardless of refine.
; options
: Optional. For weapons/armours, use this to specify specific enchants, random options, or cards compounded. This does a very simple string match for the equipment's option piece, and will not work for any complicated enchant combinations.
; placeholder
: Optional. Before data is retrieved, this text will be displayed. Defaults to <code>...</code>.
; notfound
: Optional. If no one is currently selling the item requested, this controls what is displayed. Defaults to <code>N/A</code>.
