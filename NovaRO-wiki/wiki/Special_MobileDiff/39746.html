DifficultyTickets show the reset time and tickets for up to two difficulty of an instance.
 
== Syntax ==
 
<pre>
{{DifficultyTickets|
difficulty1=|reset1=|days1=|hours1=|soloticket1=|partyticket1=|
difficulty2=|reset2=|days2=|hours2=|soloticket2=|partyticket2=
}}</pre>
 
; Parameters
; difficulty1 : The name of the difficulty (or leave it empty for normal mode).
; reset1 : The cooldown or reset time for the instance.
; days1 : '''(Optional)''' If the reset is longe than one day you should use this, with '''reset1''', to show the cooldown.
; hours1 : '''(Optional)''' Shows the hours of the cooldown, should only be used if you are using '''days1'''.
; soloticket1 : Show the amount of tickets are available for solo players.
; partyticket1 : Show the amount of tickets are available for partys.
; difficulty2 : The name of the difficulty (or leave it empty for normal mode).
; reset2 : The cooldown or reset time for the instance.
; days2 : '''(Optional)''' If the reset is longe than one day you should use this, with '''reset2''', to show the cooldown.
; hours2 : '''(Optional)''' Shows the hours of the cooldown, should only be used if you are using '''days2'''.
; soloticket2 : Show the amount of tickets are available for solo players.
; partyticket2 : Show the amount of tickets are available for partys.
 
== Example ==
 
Example for a dungeon with 2 difficulties:
*normal resets at 0:00, 2 tickets for solo, 4 tickets for party
*hard has a cd of 2 days and 4 hours, 1 tickets for solo, 5 tickets for party
{{DifficultyTickets|
difficulty1=|
reset1=0:00|
days1=|
hours1=|
soloticket1=2|
partyticket1=4|
difficulty2=|
reset2=|
days2=2|
hours2=4|
soloticket2=1|
partyticket2=5
}}
 
Example for a dungeon with 2 difficulties but with custom names for it:
*Same example as before but this time the instances have custom names for each mode
 
{{DifficultyTickets|
difficulty1=Pew Pew Everything Mode|
reset1=0:00|
days1=|
hours1=|
soloticket1=2|
partyticket1=4|
difficulty2=This Is Fun Mode|
reset2=|
days2=2|
hours2=4|
soloticket2=1|
partyticket2=5
}}
