NewNaviLink is used to automatically copy the navi command with the map name and its coordinates when the text is clicked.
 
== Syntax ==
 
<code><nowiki>{{NewNaviLink|map=|text=}}</nowiki></code>
 
; Parameters
; map : The name of the map, it's the name that is involved in parenthesis.
; text : Optional. Text to show instead of the default text <code>(<map> <x>,<y>)</code>.
 
To get the name of the map and the coordinates you can use <code>/where</code> inside a map.
 
== Example ==
Let's use an example of Abyss Lake, typing <code>/where</code> there we get:
 
[[File:Where-example.png]]
 
The name of the map is '''abyss_01'''
 
With all the data in our hands, using <code><nowiki>{{NewNaviLink|map=abyss_01}}</nowiki></code> will result into: {{NewNaviLink|map=abyss_01}}.
 
Alternatively, using <code><nowiki>{{NewNaviLink|map=abyss_01|text=somewhere in Abyss Lake 1F}}</nowiki></code> will result into: {{NewNaviLink|map=abyss_01|text=somewhere in Abyss Lake 1F}}.
 
Clicking the text will copy the navi command into the clipboard, which can then be pasted using <code>CTRL+V</code> in game.
</noinclude>
