Syntax:
Skill is used to make a plain link to the [https://www.divine-pride.net/ Divine Pride] skill page.
<code><nowiki>{{item |id=<id>}}</nowiki></code>
 
== Syntax ==
 
<code><nowiki>{{Skill|id= Name of the Skill}}</nowiki></code>
 
; Parameter
; id : The unique ID of the skill.
; Name of the Skill : The name of the skill, this is not a named parameter so you don't need to use the <code>=</code> sign.
 
== Example ==
 
Using <code><nowiki>{{Skill|id=1 Basic Skill}}</nowiki></code> will result into: {{Skill|id=1 Basic Skill}}
</noinclude>
