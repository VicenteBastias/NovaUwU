Line of skills to be used in [[Template:Skill Tree]].
 
== Syntax ==
<code><nowiki>{{Skill_Tree_Line|bottom=|short names=|class=|id1=|skill name1=|id2=|skill name2=|id3=|skill name3=|id4=|skill name4=|id5=|skill name5=|id6=|skill name6=|id7=|skill name7=}}</nowiki></code>
 
; Parameters
; bottom: '''(Optional)''' If you want names below skill icons instead of the default option at the top
:'''yes''' if you want it on
; short names: '''(Optional)''' If you want shorter names similar to how it's displayed in game
:'''on''' if you want it on
; class: class of the template used (don't forget to use <code>_</code> when needed instead of a space)
; id: the id of the skill to show
; skill name: name of the skill to display
; skill level: '''(Optional)''' to show the level of the skill
; max level: '''(Optional)''' to show the max level of the skill
 
Supports up to 7 id and 7 skill names of the same template. Check [[Template:Skill Tree]] for more info.
 
Not all slots need to be used, any can be left empty.
