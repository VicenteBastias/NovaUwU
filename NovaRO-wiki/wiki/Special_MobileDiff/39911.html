Template used to include a step by step instruction about which option the player has to choose when using the '''Warper''' NPC.
== Syntax ==
 
<code><nowiki>{{Warper}}</nowiki></code> will produce {{Warper}}
 
<code><nowiki>{{Warper|Dungeons}}</nowiki></code> will produce {{Warper|Dungeons}}
 
<code><nowiki>{{Warper|Dungeons|Abyss Lakes}}</nowiki></code> will produce {{Warper|Dungeons|Abyss Lakes}}
 
=== Parameters ===
 
As you can see there is no need to name the parameters, but the order is important and remember to capitalize it correctly.
For now we can have 10 parameters at maximum, if there comes a time that more is needed just add <code><nowiki>{{#if:{{{11|}}} | &amp;ndash;&amp;rsaquo; {{{11}}}}}</nowiki></code> right before the final '''bolding''' markup.
 
== Example ==
 
Using <code><nowiki>{{Warper|Dungeons|Glast Heim|Glast Heim St. Abbey}}</nowiki></code> will result into {{Warper|Dungeons|Glast Heim|Glast Heim St. Abbey}}
 
 
</noinclude>
