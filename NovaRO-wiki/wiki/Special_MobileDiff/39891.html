Stats is used to describe what stats do in each class.
 
== Syntax: ==
<pre>{{Stats|str=|agi=|vit=|int=|dex=|luk=}}</pre>
 
If no info is added, the table will fetch basic info that says that something needs to be added. This was made to be easier to players add info to the class guide pages.
 
== Parameters: ==
; '''str''' : description of what STR does to your class
; '''agi''' : description of what AGI does to your class
; '''vit''' : description of what VIT does to your class
; '''int''' : description of what INT does to your class
; '''dex''' : description of what DEX does to your class
; '''luk''' : description of what LUK does to your class
 
== Example: ==
 
Using
<pre>
{{Stats|
str=I boost your weight|
agi=I increase your spam!|
vit=Health points it's with me!|
int=You need to be smart, add more points here!|
dex=|
luk=I wanted to provide luck to your drops but the cake was a lie}}</pre>
will provide the following table below where someone forgot to add info to what dex does to the class.
 
{{Stats|
str=I boost your weight|
agi=I increase your spam!|
vit=Health points it's with me!|
int=You need to be smart, add more points here!|

luk=I wanted to provide luck to your drops but the cake was a lie}}
