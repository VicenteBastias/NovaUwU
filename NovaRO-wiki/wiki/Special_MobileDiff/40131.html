Align, as the name says, is used to align content in the page. Use it only as a last resort if you can't align the content with MediaWiki syntax.
 
== Syntax ==
<code><nowiki>{{Align|param1|param2}}</nowiki></code>
 
; Parameters
; ''param1'' : The first parameter should be the position in the page you want the content to be aligned at. You can choose between: '''center''', '''left''' and '''right'''.
; ''param2'' : The second parameter should be the content that you want to align.
 
If the template messes up your page layout you should follow it with a <code><nowiki><br style="clear: both;" /></nowiki></code> tag.
 
== Example ==
 
Using <code><nowiki>{{Align|left|Content to be aligned.}}</nowiki></code> will result into:
 
{{Align|left|Content to be aligned.}}<br style="clear: both;" />
 
Using <code><nowiki>{{Align|center|Content to be aligned.}}</nowiki></code> will result into:
 
{{Align|center|Content to be aligned.}}
 
Using <code><nowiki>{{Align|right|Content to be aligned.}}</nowiki></code> will result into:
 
{{Align|right|Content to be aligned.}}
</noinclude>
