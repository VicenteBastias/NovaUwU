NPCBox is used to give a quick summary of information about an NPC. The table will be positioned at the right portion of the page.
 
== Syntax ==
 
<code><nowiki>{{NPCBox|name=|image=|purpose=|map=|coordinates=|bgColor=}}</nowiki></code>
 
; Parameters
; name : The name of the NPC.
; image : The name of the image file that was uploaded to the wiki, you don't need to specify the file extension but the image needs to be uploaded as PNG.
; purpose : A little description of the NPC main function, keep it simple.
; map : The name of the map the NPC is located at, you can get that using the command <code>/where</code> in-game.
; coordinates : The location of the NPC, you can get that using the command <code>/where</code> in-game. Recommended to use [[Template:NaviLink]] in here.
; bgColor : '''(Optional)''' Changes the title background color, if it's not specified the default will be used.
;: '''bgColor''' accept colors value according to the W3C [https://www.w3schools.com/colors/default.asp CSS Colors] specification.
 
== Example ==
 
Using <code><nowiki>{{NPCBox|name=Harek Ignison|image=HarekIgnison|purpose=Shadow Blacksmith|map=The Wyvern (@go mh)|coordinates={{NaviLink|map=2009rwc_03|x=245|y=194}}}}</nowiki></code> will result into:
 
{{NPCBox|name=Harek Ignison|image=HarekIgnison|purpose=Shadow Blacksmith|map=The Wyvern (@go mh)|coordinates={{NaviLink|map=2009rwc_03|x=245|y=194}}}}
</noinclude>
