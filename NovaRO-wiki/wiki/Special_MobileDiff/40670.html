Shows the item thumbnail and item name that links to the [https://www.novaragnarok.com/?module=vending NovaRO Market] database.
Same as [[Template:Item List]], but always pull the item thumbnail from the alternative database (<code><nowiki>https://novaragnarok.com/data/items/icons2/<id>.png</nowiki></code> instead of <code><nowiki>https://novaragnarok.com/data/kRO/inventory/<id>.png</nowiki></code>).


If thumbnail is missing, try [[Template:Item List]].
If thumbnail is missing, try using [[Template:Item List]] without <code>alt=yes</code>.
 
For showing items without a thumbnail, use [[Template:Item]] (links to item overview) or [[Template:Market]] (links to live market data).
 
For cards, use [[Template:Card]]
 
(Note: This page no longer links to Divine Pride. Huge thanks and credit to the iROwiki guys! AND THANKS TOKEI.)
 
== Syntax ==
 
<code><nowiki>{{Item List|id=|item=|num=|slots=|link=}}</nowiki></code>
 
; Parameters
; id : The unique id of the item.
; item : The name of the item.
; num : '''(Optional)''' The quantity of items.
; evt : '''(Optional)''' If the item is a version exclusive to Endeavor Tokens and the user wants to add the tag to the item. Use <code>evt=yes</code> for that.
; gc : '''(Optional)''' If the item is a version exclusive to Gold Coins and the user wants to add the tag to the item. Use <code>gc=yes</code> for that.
; slots : '''(Optional)''' The number of slots of the item has it.
; link : '''(Optional)''' Links to the price list section instead of the view section if equals to 1.
; prefix : '''(Optional)''' Prefix (e.g. card prefixes) added in front of the item name (after num).
; suffix: '''(Optional)''' Suffix (e.g. card suffixes) added behind the item name (before slots).
 
== Example ==
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer}}
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer|num=5}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer|num=5}}
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer|slots=2}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer|slots=2}}
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer|slots=2|link=1}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer|slots=2|link=1}} that opens the selling tab
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer|evt=yes|slots=1}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer|evt=yes|slots=1}}
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer|gc=yes|slots=1}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer|gc=yes|slots=1}}
 
Using <code><nowiki>{{Item List|id=2285|item=Apple of Archer|gc=yes|slots=2|prefix=Cursed|postfix=of WTF}}</nowiki></code> will result into {{Item List|id=2285|item=Apple of Archer|gc=yes|slots=2|prefix=Cursed|suffix=of WTF}}
</noinclude>
