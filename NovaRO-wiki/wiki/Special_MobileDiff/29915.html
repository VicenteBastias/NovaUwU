GuideBox is used to give basic information about the class the guide refers, the author name and the job stats bonus.
 
== Syntax ==
 
<code><nowiki>{{GuideBox|job=|image=|basejob=|author=|str=|agi=|vit=|int=|dex=|luk=}}</nowiki></code>
 
; Parameters
; job : The job that the guide is about.
; image : The name of image file that was uploaded to the wiki, the image needs to be a PNG.
; basejob : The name of the first job of the class.
; author : The author name.
; str : The STR job bonus of the class.
; agi : The AGI job bonus of the class.
; vit : The VIT job bonus of the class.
; int : The INT job bonus of the class.
; dex : The DEX job bonus of the class.
; luk : The LUK job bonus of the class.
 
== Example ==
 
Using <code><nowiki>{{GuideBox|job=Guillotine Cross|image=Guillotinecross|basejob=Thief|author=WikiAuthor|str=8|agi=16|vit=6|int=5|dex=9|luk=4}}</nowiki></code> will result into:
 
{{GuideBox|job=Guillotine Cross|image=Guillotinecross|basejob=Thief|author=WikiAuthor|str=8|agi=16|vit=6|int=5|dex=9|luk=4}}
</noinclude>
