Main Page Nav is used in [[Template:Main Page Menus]] to create a navigation menu.
 
== Syntax ==
 
<code><nowiki>{{Main Page Nav|index-X-link|index-X-text|index-X-image}}</nowiki></code>
 
Where '''X''' goes from 1 to 10.
 
; Parameters
; index-X-link : The link to the target page.
; index-X-text : Usually the name of the target page.
; index-X-image : An image to distinguish the destination.
