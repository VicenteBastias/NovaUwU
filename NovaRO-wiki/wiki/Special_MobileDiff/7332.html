=== Equipment Sets ===
These are items that need to be worn in a full set for them to really shine.  These items have the benefit of being cheap and can last you until max level if you are unable to replace equipment.
* Gray Set: {{Item List|id = 18820|item = Gray Helmet [1]|simple = yes}}, {{Item List|id = 15091|item = Gray Robe [1]|simple = yes}}, {{Item List|id = 20721|item = Cloak of Gray [1]|simple = yes}} and {{Item List|id = 22033|item = Boots of Gray [1]|simple = yes}}
Each piece of the set is purchased from the [[Ghost Palace]] instance using Gray Shards.  Combined, the set offers you +25% Increased HP and SP, +15% Neutral Resistance and +10% MATK.  The HP, SP and Neutral Resistances make this set one of the best tanking sets in the game for Sura and a handy leveling set.  Each item also offers holy resistance which can be helpful in some situations.  Each piece of equipment having a slot means that you can enhance the bonuses even more or use the slots to help with immunity to statuses.  I recommend these cards depending on the item, with a brief explanation for why:
* {{Item List|id = 18820|item = Gray Helmet [1]|simple = yes}}:  {{Item List|id = 4269|item = Incubus Card|simple = yes}} to use with a {{Item List|id = 4218|item = Succubus Card|simple = yes}}, {{Item List|id = 4127|item = Nightmare Card|simple = yes}} for sleep immunity if required or {{Item List|id = 4582|item = Bungisngis Card|simple = yes}} if refined for HP.
* {{Item List|id = 15091|item = Gray Robe [1]|simple = yes}}: {{Item List|id = 4218|item = Succubus Card|simple = yes}} for health, {{Item List|id = 4031|item = Peco Peco Card|simple = yes}} for even more health or {{Item List|id = 4105|item = Marc Card|simple = yes}} for when you're fighting bosses or regular enemies with freeze.
* {{Item List|id = 20721|item = Cloak of Gray [1]|simple = yes}}: {{Item List|id = 4133|item = Raydric Card|simple = yes}} or {{Item List|id = 4334|item = Noxious Card|simple = yes}} for damage reduction.
* {{Item List|id = 22033|item = Boots of Gray [1]|simple = yes}}: {{Item List|id = 4381|item = Green Ferus Card|simple = yes}} or {{Item List|id = 4107|item = Verit Card|simple = yes}} for survivability.
