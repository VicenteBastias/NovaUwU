== Syntax ==
 
<code><nowiki>{{BaseStat|stat=|number=}}</nowiki></code>
 
; Parameters
; stat : The name of the stat you want to refer. Usually abbreviated like: '''STR'''.
; number : The quantity of the base stat.
 
== Example ==
 
Using <code><nowiki>{{BaseStat|stat=STR|number=120}}</nowiki></code> will result into: {{BaseStat|stat=STR|number=120}}
