Template:Enemies2

  • Language
  • Watch
  • Edit

Enemies2 is used to create a table row to show information about a monster. This is the same as Template:Enemies but also has a column for amount of monsters presented in a map.

This template SHOULD NOT be used outside a table.

To save you from having to write the header over and over you can use it in conjunction with Template:MobTable2, in the Example you will see how.

Syntax

{{Enemies|id=ID|name=Name|hp=HP|level=Level|size=Size|race=Race|element=Element}}

Parameters
id
The unique identifier number that refers to the monster.
name
The name of the monster, it should be capitalized.
hp
The hit points value, use commas as the thousands separators. Like so: 2,230,000
level
The level of the monster.
size
The size of the monster.
race
The race of the monster.
element
The element of the monster, it should be accompanied by it's element level.
amount
The number of monsters in a certain map.

Example

{{MobTable2|data=
{{Enemies2|id=1002|name=Poring|hp=55|level=1|size=Medium|race=Plant|element=Water 1|amount=23}}
{{Enemies2|id=1063|name=Lunatic|hp=48|level=3|size=Small|race=Brute|element=Neutral 3|amount= 0}}
}}

will result into:

Image Name Level HP Size / Race / Element Amount
Poring 1 55 Medium / Plant / Water 1 23
Lunatic 3 48 Small / Brute / Neutral 3 0
