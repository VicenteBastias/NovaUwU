Enchanted is used to tell about the possibility of the gear being enchanted and provide a link to it.

Syntax

{{Enchanted|link=Page URL}}

Parameters
link
The page URL that has the information about the enchantment process.

Example

Using {{Enchanted|link=[[Illusion Equipments and Enchants (17.1)#Equipment Enchanting|Illusion Enchants]]}} will result into:

Can be enchanted - Illusion Enchants.
