NewNaviLink is used to automatically copy the navi command with the map name and its coordinates when the text is clicked.

Syntax

{{NewNaviLink|map=|text=}}

Parameters
map
The name of the map, it's the name that is involved in parenthesis.
text
Optional. Text to show instead of the default text (<map> <x>,<y>).

To get the name of the map and the coordinates you can use /where inside a map.

Example

Let's use an example of Abyss Lake, typing /where there we get:

The name of the map is abyss_01

With all the data in our hands, using {{NewNaviLink|map=abyss_01}} will result into: abyss_01Copied!.

Alternatively, using {{NewNaviLink|map=abyss_01|text=somewhere in Abyss Lake 1F}} will result into: somewhere in Abyss Lake 1FCopied!.

Clicking the text will copy the navi command into the clipboard, which can then be pasted using CTRL+V in game.
