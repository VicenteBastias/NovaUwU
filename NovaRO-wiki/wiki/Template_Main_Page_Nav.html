Main Page Nav is used in Template:Main Page Menus to create a navigation menu.

Syntax

{{Main Page Nav|index-X-link|index-X-text|index-X-image}}

Where X goes from 1 to 10.

Parameters
index-X-link
The link to the target page.
index-X-text
Usually the name of the target page.
index-X-image
An image to distinguish the destination.

Example

Using 

{{Main Page Nav
| index-1-link = Leveling Guide
| index-1-text = Leveling
| index-1-image = Mainwoe.png

| index-2-link = Frequently Asked Questions
| index-2-text = FAQ
| index-2-image = Mainfaq.png

| index-3-link = Class Guides
| index-3-text = Class Guides
| index-3-image = Mainclass.png

| index-4-link = Nova Shop
| index-4-text = Nova Shop
| index-4-image = Maincash.png

| index-5-link = Troubleshooting
| index-5-text = Troubleshooting
| index-5-image = Maintrouble.png

| index-6-link = Rules
| index-6-text = Rules
| index-6-image = Mainrules.png

| index-7-link = Instances
| index-7-text = Instances
| index-7-image = Maininstance.png

| index-8-link = Custom Headgear Quests
| index-8-text = Headgears
| index-8-image = 20455.png

| index-9-link = Monster Hunter Portal
| index-9-text = Monster Hunter
| index-9-image = Carver.png
}}

will result into:

Leveling

FAQ

Class Guides

Nova Shop

Troubleshooting

Rules

Instances

Headgears

Monster Hunter
