Crit Rifle Spammer (Spiral Shooting & Only One Bullet with Rifle)

Notes

  • Every rifle skill crits
  • 1.5 sec FCT needed
  • Ideal usage is alternating Spiral Shooting and Only One Bullet
  • 70~72% ACD for alternating both skills and as much ASPD as possible
    • Why? Faster spam and better DPS than spamming 1 thing

Testing Notes

Due to the recent Atk % bug change, some intensive testing was done on the existing gear in NovaRO to verify BiS armor, headgear, and shoe combination. PLEASE NOTE: THIS TEST WAS BEFORE EP 18 SO GRAY WOLF GEAR AND EVENTUAL HERO BOOTS ENCHANTS WILL MOST LIKELY BE BIS FOR MOST BUILDS

Testing Notes

  • This was the base build that was used for testing (only changed armor, hat, and shoe based on test): https://imgur.com/a/oJylP3X
  • The dmg is only for ONE tick so multiply by 7 for full dmg of 1 vigilante
  • Only buff used was ultimate and no consums
  • This was on minimum variance

Some findings based on test

  • Illusion sprint set is very good for boss and can compete with AAA combo with double proc. Big downside is that u only get 0.7 sec FCT and 12% ACD so for minmaxing, it might lose in long run since you'll sacrifice other gear to make ACD+FCT requirement
  • AAA combo with double procs still is superior build (at the time of this testing)
  • Chung cap is honestly not a bad choice when fighting non-large monster considering how cheap it is and also gives bonus FCT to make life easier
  • If goal is long term, better to start saving up for hero boots-LT if you already plan to get the black-LT since +11 C grade boots are already beating no proc +11 auto legs but +13 grade A gives u 1 sec FCT flexibility and will be enchantable for more dmg soon^tm
