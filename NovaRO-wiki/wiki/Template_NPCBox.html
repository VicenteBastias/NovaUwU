Template:NPCBox

  • Language
  • Watch
  • Edit

NPCBox is used to give a quick summary of information about an NPC. The table will be positioned at the right portion of the page.

Syntax

{{NPCBox|name=|image=|purpose=|map=|coordinates=|bgColor=}}

Parameters
name
The name of the NPC.
image
The name of the image file that was uploaded to the wiki, you don't need to specify the file extension but the image needs to be uploaded as PNG.
purpose
A little description of the NPC main function, keep it simple.
map
The name of the map the NPC is located at, you can get that using the command /where in-game.
coordinates
The location of the NPC, you can get that using the command /where in-game. Recommended to use Template:NaviLink in here.
bgColor
(Optional) Changes the title background color, if it's not specified the default will be used.
bgColor accept colors value according to the W3C CSS Colors specification.

Example

Using {{NPCBox|name=Harek Ignison|image=HarekIgnison|purpose=Shadow Blacksmith|map=The Wyvern (@go mh)|coordinates={{NaviLink|map=2009rwc_03|x=245|y=194}}}} will result into:

Harek Ignison
NPC Purpose: Shadow Blacksmith
Map: The Wyvern (@go mh)
Coordinates: (2009rwc_03 245, 194)Copied!
