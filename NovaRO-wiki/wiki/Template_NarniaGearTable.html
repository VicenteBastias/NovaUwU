Template:NarniaGearTable

From NovaRO: Wiki
Jump to navigation Jump to search

Used to create a table specializing in displaying gear with 'Template:NarniaGear. Used by Lady Narnia.

  • Flexible
  • Easy-to-use
  • Optimized for both desktop and mobile screens
  • Automatically injects table into the Table of Contents using header hack.
  • Must be used with Template:NarniaGear for complete functionality.

Syntax

{{NarniaGearTable |class= |style= |title= |gears= }}


Parameters
season (optional)
Colors the table to match the holidays. Season is globally set using Template:NarniaHoliday
Current options: auto, halloween, christmas
class (Optional)
Classes can be added here for styling, functionality, and/or other features. (mw-collapsed, wikitable, etc.)
style (Optional)
Put CSS styles here to customize the table to your liking. (background-color, font-size, width, etc.)
title (Optional)
The title of the table. If empty, defaults to "Gears".
gears
Only accepts gears in the form of Template:NarniaGear templates.
