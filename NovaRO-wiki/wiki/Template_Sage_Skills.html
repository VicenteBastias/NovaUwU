Abracadabra Support
  • Max Level: 10
  • Target: Self
  • SP Cost: 50
  • Consumes: 2  Yellow Gemstone
 Abracadabra allows you to cast random skills, even some exclusive ones as well.

The skills cast are equal to the level of this skill.

Below is a list of special skills only available through this skill.


  • This skills will always use at least 1  Yellow Gemstone, even if you have effects that remove gemstone consumption.

Monocell Change a monster into a Poring for an amount of time.


  • This skill shows up as "????" when choosing a target.

Class Change Change a monster into a random field MvP for an amount of time.


  • This skill shows up as "????" when choosing a target.

Monster Chant Summons a random monster. (Similar to  Dead Branch)

Grampus Morph Change the caster's face to be SUPER UGLY for a period of time.

Grim Reaper Instantly kills a monster.


  • You receive no experience for the monster killed.
  • This does not work on boss monsters.
  • This skill shows up as "????" when choosing a target.

Gold Digger Steals zeny from a monster.


Amount of zeny is determined using this formula;
Monster Level * 100

  • This skill shows up as "????" when choosing a target.

Beastly Hypnosis Allows you to tame any tamable monster without the taming consumable.

  • This also works on the White Lady MvP.
  • This skill shows up as "????" when choosing a target.

Questioning The caster uses the /? emote.

Gravity Shows the Gravity logo above the caster's head.

Leveling Gives you 10% of the experience needed for the next level.

Suicide Instantly kill the caster.

Rejuvenation Fully heal the caster's HP and SP.

Coma Set the caster's HP and SP to 1.
