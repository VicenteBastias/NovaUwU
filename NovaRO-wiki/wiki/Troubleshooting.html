Upon encountering any errors, please make sure to do all the steps listed below.

  • Always use the Full Installer. You can download it here.
  • Right-click on NovaRO.exe and set its Compatibility mode to Windows 7, if you're running Windows 10 or Windows XP Service Pack 3, if you're running Windows 7.
  • Right-click on NovaRO.exe and tick the box next to Run as Administrator.
  • Right-click on Nova Patcher.exe and tick the box next to Run as Administrator.
  • Be sure to grab the latest patch through Nova Patcher.exe
  • Avoid installing the game to Program Files. Try to install it to somewhere like C:\Games\
  • Disable Anti Virus AND Firewall software before installing and/or patching. (Especially in the case of Avast and McAfee)
  • If you use Warsaw (a bank software popular in Brazil), you will need to uninstall it.

Should you still have any issues, please explain your problem on Discord's #technical-support channel.

Installer issues

The installer doesn't start

  1. Disable your antivirus; it might be scanning the whole file.
  2. Depending on your computer, it can take some time to start; wait patiently.

The install stops at "data.grf"

  • Data.grf is the biggest file in the installer; the installer is working, but it doesn't show the progress (wait a few minutes).

The installer cannot connect or takes too long

  1. There is a connection issue to the download server when downloading the BGM folder through the installer.
  2. Using a VPN will most likely get rid of the issue. You can try SoftEther, for free, by clicking here.


Patcher issues

START GAME button does not open the game

  • The game will still appear on your Task Manager as running.
  1. Close your Nova Patcher.exe if it is still open.
  2. Right-click Nova Patcher.exe > Properties.
  3. Select the Compatibility tab (second tab).
  4. Set its Compatibility mode to Windows 7, if you're running Windows 10 or Windows XP Service Pack 3, if you're running Windows 7.
  5. Check Run this program as administrator.
  6. Click Apply, then OK.
  7. Open Nova Patcher.exe and the game should open as normal.

"Your game executable has been modified or is not the latest version"

  • Problem: There might be an issue with your client version, re-install NovaRO.
  • Problem: You have two server clients in the same folder.
    Solution: Delete the folder and install the two servers to two different folders.
  • Problem: You are running two different server's clients at the same time.
    Solution: NovaRO can be only run with other NovaRO clients. Try closing the other clients to make it work.
  • Problem: Your game is not up to date.
    Solution: Run the Nova Patcher.exe and wait until it patches.
  • Problem: Your game is not up to date, but you patched.
    Solution: Disable your firewall/antivirus during the patching or add the exe as an exception.

"Impossible to read more data from the download stream..."

  1. Right-click the background > Retry.
  2. This error shows up when a file fails to download.
  3. If the file appears to be large, look at the name when it is downloading.
    Download the patch file manually by typing the following URL:
    https://www.novaragnarok.com/patch/data/name_of_the_file.thor
  4. Place the file in your NovaRO folder and run the patcher again.

".NET Framework 3.5 is missing"

  1. Download the following files
    • https://www.microsoft.com/en-ca/download/details.aspx?id=21
    • https://www.microsoft.com/en-ca/download/details.aspx?id=17851
  2. Install both of the frameworks (if it's already installed it will tell you).

Patcher only shows a white bar, I can't do anything

  1. Download the latest patcher version here (https://www.novaragnarok.com/download/Patcher.zip) and extract the files into your NovaRO folder.
  2. Run Patcher.exe again.

I can't change my patcher background/My patcher window is small

This is due to your computer not supporting a high resolution. You can force it to go bigger by placing this tkextra.ini file into your NovaRO folder.

  • You can remove it anytime by deleting it from your folder.


Client issues

Cannot init d3d or grf file has problem

  • Ragnarok has a maximum resolution of 2048 x 1536. This cannot be changed due to Ragnarok being based off of Direct X7.
  • Ensure your monitor's native resolution is set to 2048 x 1536 or lower. On some machines, this requires a computer restart.
  • Alternatively, you can edit config.ini in the NovaRO folder and set the width and height to lower than 2048 x 1536, and fullscreen_mode: 0

Crashing in certain maps as well as from Fire Rain

  • Example maps: pay_dun01, iz_dun03, gef_dun00, beach_dun.
  • The following steps can also be used to fix an issue with the game not starting after pressing the "Start Game" button on the Nova Patcher.
  • Note, this will not work on Windows XP.
  1. Close your NovaRO.exe if it is still open.
  2. Right-click NovaRO.exe > Properties.
  3. Select the Compatibility tab (second tab).
  4. Check Run this program in compatibility mode for Windows XP (Service Pack 3).
  5. Check Run this program as administrator.
  6. Click Apply, then OK.
  7. Open NovaRO.exe and enjoy playing.

Cannot find File: ...

  1. Exit the game.
  2. Run the NovaRO patcher and patch your game.

"Ragnashield.dll is missing"

  1. Disable your firewall and antivirus.
  2. Delete the NovaRO folder.
  3. Run the NovaRO Installer again.

My client auto-close randomly

  1. Download the latest client version over here:
    https://novaragnarok.com/download/clients/NovaRO_20210819.zip

I can't click anything, I can't move. I can write in channels though.

  1. You are having issues with the new client, download the old client version over here: https://www.novaragnarok.com/download/clients/NovaRO_201701.zip and extract to your NovaRO folder.

The procedure entry point MiniDumpWriteDump could not be located in the dynamic link library dbghelp.dll

  1. Open your NovaRO folder and rename the file "dbghelp.dll" to "dbghelp_old.dll" or anything else.
    This will make you use the Windows's version of the DLL instead of the one provided by default.

The procedure entry point MiniDumpWriteDump could not be located in the dynamic link library nvspcap.dll

  1. Open your NovaRO folder and rename the file "dbghelp.dll" to "dbghelp_old.dll" or anything else.
    This will make you use the Windows's version of the DLL instead of the one provided by default.

When I use a skill or attack, I get an 'attempt to call a nil value'/'IsItemUsingArrow' error.

Solution 1

  1. Simply run your Nova Patcher Alternative.exe from your NovaRO folder and the issue should be resolved.

Solution 2

  1. Open your patcher, right-click the background and select Reset all patches.
  2. Then right-click the background again and use Retry.
  3. Let it patch and you should be good to go.

Solution 3

  1. Download the Full Installer and patch up.

I see missing sprites ingame.

Solution 1

  1. Simply run your Nova Patcher Alternative.exe from your NovaRO folder and the issue should be resolved.

Solution 2

  1. Open your patcher, right-click the background and select Reset all patches.
  2. Then right-click the background again and use Retry.
  3. Let it patch and you should be good to go.

Solution 3

  1. Download the Full Installer and patch up.

I can only see RGB Emulation

  1. This fix only applies if you only see RGB emulation in your Setup.exe's list of Graphic Devices.
  2. Simply uncheck "Disable hardware-accelerated graphics" and it should give you new options in the list of Graphic Device.
  3. Select the first option in Graphic Device (which should be Direct3D HAL).

I want to play on a Mac / Linux operating system

  • Mac and Linux support is in beta.
  • Here's a tutorial for running NovaRO on Mac:
    • https://gist.github.com/phaicm/cd7b66b82232b1535fbd2652d6d048e3
  • And here's the Linux one:
    • https://gist.github.com/phaicm/cc6c876c903063720617b3b3e344ffcc
  • Alternatively there are 2 other options to play NovaRO on a Mac or Linux machine:
    1. Virtual Machine: Install a Windows OS on a virtual machine. Example of virtual machine software includes Virtual Box, VMware, Parallels, etc.
    2. Dual Booting: Install a Windows OS on a separate partition of your hard disk. This can be done using partition management software, such as Boot Camp or GParted.

    • https://aka.ms/vs/15/release/vc_redist.x86.exe
    • https://aka.ms/vs/17/release/vc_redist.x86.exe

My client closes without any warning

  1. This is most likely due to a process named core.exe (Warsaw) that identifies NovaRO as a virus and closes it.
  2. To stop Warsaw:
    1. Use Windows > Run > "services.msc". Then disable all GAS Tecnologia services.
    2. The service may turn itself back on without your consent, in which case you'll have to disable it again or look at other ways to fully remove it.

MSVCP140.dll is missing

WMI related issues

There are multiple errors related to WMI. Here are the most common ones.

Using Windows XP (RagnaShield Beta - WMI generic failure)

  1. Start menu, Run...
  2. Type services.msc, press enter.
  3. Scroll down and look for Windows Management Instrumentation.
  4. Enable it.

Error code: 0x80070422

  1. Start menu, Run...
  2. Type services.msc, press enter.
  3. Scroll down and look for Windows Management Instrumentation.
  4. Right-click > Properties > Set the Startup type to Automatic.
  5. Click Start, then OK.

Error code: 0x80041002 (or 0x80041013, 0x80041010)

Solution 1

  1. Run wmi_rebuild.bat in your NovaRO folder with right-click > Run as Admin.
  2. After it finishes, restart your computer.
  3. If you get errors while doing this (or if it didn't help), go to solution 2.

Solution 2

  1. Start menu, Run...
  2. Type services.msc, press enter.
  3. Scroll down and look for Windows Management Instrumentation.
  4. Right-click > Properties > Set the Startup type to Disabled.
  5. Reboot your computer (this step is required).
  6. Go in C:\Windows\System32\wbem\
  7. Rename repository to repository.old.
  8. Open the Services window (step 1 to 3).
  9. Right-click > Properties > Set the Startup type to Automatic.
  10. Click Start, then OK.


New 2018 client update

You receive a Win32 application error

  1. This problem is caused by a bug in the default patcher.
  2. You can redownload the 2018 files manually from https://www.novaragnarok.com/download/clients/NovaRO_201809.zip
  3. If that doesn't help, you can revert back to the 2017 legacy client (https://www.novaragnarok.com/download/clients/NovaRO_201701.zip).
  4. Alternatively, the updated version of the Nova Patcher (https://www.novaragnarok.com/download/NovaPatch.zip) will be able to download this new client without any issues.

This app can't run on your PC (Windows 10)

  1. Redownload the 2018 files manually from https://www.novaragnarok.com/download/clients/NovaRO_201809.zip
  2. If that doesn't help, you can revert back to the 2017 legacy client (https://www.novaragnarok.com/download/clients/NovaRO_201701.zip).


My screen looks like this how do I fix it?

  1. Make sure your graphic drivers and directX are up to date.
  2. Make sure you have the correct graphic device selected in your setup.exe


Ingame issues

Screenshot: PrntScr button doesn't work

  1. Press Esc > Options and Settings > Interface.
  2. Change the hotkey for "Screenshot".

Screenshot: The image is grey/distorted

  1. Run NovaRO\Setup.exe.
  2. Change your horizontal resolution to be divisible by 4.
  3. (You must do it for the width and height in NovaRO > RagnaShield > config.ini if you're using false fullscreen.)

Hotkeys are not being saved

  1. Download the [1] and extract it to your NovaRO folder.
  2. If that didn't work, download the one 2017 client and extract it to your NovaRO folder.

Failed to Connect to Server

  1. Make sure the server is not on maintenance and ask someone else on Discord if they can log in first.
  2. Try a fresh reinstall in a new folder (changing the installation drive seems to have helped someone?).
  3. Make sure the following ports are opened: 6954, 6175 and 5175.
  4. Contact a GM or require more help on the forums.

The font is too small!

  1. Open NovaRO\RagnaShield\import_config.ini and edit the values in the file.
  2. Uncomment the //Font.Size: 15 line by removing the // at the beginning.
  3. Do the same for the Font.Family (you can choose any of your liking).
  4. We recommend the following fonts:
    • Verdana 12 for no confusing I and i.
    • Arial 13 for bigger fonts.

I can't add skill points after a skill reset!

You must remove all items that grant skills before you perform a reset. Common offenders are:

  •  Heroic Backpack
  •  Silverwork Bracelet

To be on the safe side, remove all equipment before resetting.


NovaExt issues

  • Close all clients before editing NovaExt.ini.﻿
  • In NovaRO\Setup.exe, make sure "Restrict mouse to window" is unchecked.
  • Set the Sound Mode to "Use 2D Sound".
  • Install all the VC++ Redistributables.
  • Make sure your client is named NovaRO.exe (any other name will fail).
  • Make sure you are not running the client in compatibility mode. The client requires admin privileges, so check "Run this program as administrator".
  • If you still have a problem, upload your __NovaExt.log and NovaExt.ini from your NovaRO folder and state your computer specs (Intel/AMD, what graphics card, etc), Operating System, and Anti-virus programs in the following thread:
    • https://www.novaragnarok.com/forum/topic/7543-novaext-troubleshooting-problems/﻿
