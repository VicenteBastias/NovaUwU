Used in a table or Template:NarniaGearTable to add a table row specializing in displaying gear. Used by Lady Narnia.

  • Flexible
  • Easy-to-use
  • Optimized for both desktop and mobile screens
  • Best used with Template:NarniaGearTable for complete functionality.

Syntax

{{NarniaGear|style= |type= |items= |itemDesc= |notes= }}


Parameters
style (Optional)
Put CSS styles here to customize the row to your liking. Applies to both columns in the row. (background-color,font-size,etc.)
type (Optional)
Item's type/classification. (Weapon, Upper + Middle Headgear, Shadow Gear, Card, etc.)
items
Items go here. Works best when using Template:Item List2.
itemDesc (Optional)
Item descriptions and it's effects, such as INT + 1 or refine bonuses. If no effects, leave blank.
notes
Summary, suggestions, and/or recommendations of the gears listed.
