Solução de Problemas

From NovaRO: Wiki
Jump to navigation Jump to search

Ao se deparar com erros, siga todos os passos abaixo.

  • Sempre use o Full Installer (instalador completo) - o NovaRO não tem nenhum patch lite! Você pode baixar o instalador aqui.
  • Clique com o botão direito em NovaRO.exe e altere o Modo de Compatibilidade - se você estiver usando o Windows 10/8, selecione o Modo de Compatibilidade para Windows 7, e se estiver usando Windows 7, selecione o modo de compatibilidade de Windows XP Service Pack 3.
  • Clique com o botão direito em NovaRO.exe e marque a opção Executar como Administrador.
  • Clique com o botão direito em Nova Patcher.exe e marque a opção Executar como Administrador.
  • Sempre tenha certeza de que o NovaRO está atualizado, e se não estiver, execute o Nova Patcher.exe
  • Evite instalar o jogo dentro da pasta Arquivos de Programas. Ao invés disso, instale em um diretório como C:\Jogos\
  • Desative todos os Anti-vírus e Firewalls antes de instalar e/ou rodar o patcher - especialmente se você tiver Avast ou McAffee.
  • Se você tiver o Warsaw instalado (software utilizado por alguns bancos no Brasil), você terá que desinstalar.

Se você ainda estiver com problemas, vá ao NovaCord, no canal #support , e peça ajuda.

Problemas com o Instalador

O Instalador se recusa a iniciar

  1. Desative seu anti-vírus, ele pode estar escaneando o arquivo, o que impede a instalação.
  2. Dependendo do seu computador, algumas vezes pode demorar mais que o esperado.

O instalador para em "data.grf"

  • O arquivo Data.grf é o mais pesado do Ragnarök; o instalador está funcionando, mesmo que pareça parado.


Problemas com o Patcher

O botão START GAME não inicia o jogo

  • O jogo ainda aparece no Gerenciador de Tarefas.
  1. Feche o Nova Patcher.exe caso ainda esteje aberto.
  2. Clique com o botão direito em Nova Patcher.exe e então clique em Propriedades.
  3. Selecione a aba de Compatibilidade.
  4. Altere o Modo de Compatibilidade - se você estiver usando o Windows 10/8, selecione o Modo de Compatibilidade para Windows 7, e se estiver usando Windows 7, selecione o modo de compatibilidade de Windows XP Service Pack 3.
  5. Marque a opção Executar este programa como Administrador.
  6. Clique em Aplicar, e então em OK.
  7. Abra novamente o Nova Patcher.exe - o jogo deve funcionar agora.

Erro: "Your game executable has been modified or is not the latest version"

  • Problema: Seu cliente pode estar com algum problema, baixe ele novamente:
    https://www.novaragnarok.com/download/clients/NovaRO_201809.zip
  • Problema: Você tem dois clientes na mesma pasta.
    Solução: Delete a pasta, e mantenha uma pasta para cada cliente separadamente.
  • Problema: Você está executando dois clientes diferentes ao mesmo tempo.
    Solução: O NovaRO não pode ser executado em conjunto com outros servidores.
  • Problema: Seu jogo está desatualizado.
    Solução: Execute o Nova Patcher.exe e deixe atualizar.
  • Problema: Seu jogo está desatualizado, mas você deixou o Patcher atualizar.
    Solução: Desative o seu anti-vírus e/ou firewall, e se possível, adicione o Nova Patcher.exe às exceções.

Erro: "Impossible to read more data from the download stream..."

  1. Clique com o botão direito dentro do Patcher > Retry.
  2. Esse erro só aparece se algum arquivo falhou o seu download.
  3. Se o arquivo demorar muito, anote seu nome.
    E então baixe ele manualmente seguindo o modelo:
    https://www.novaragnarok.com/patch/data/nome_do_arquivo.thor
  4. Coloque o arquivo baixado na pasta do NovaRO e tente atualizar novamente.

Patches Manuais

CUIDADO: os patches devem ser instalados em ordem, e podem ocasionar problemas em seu jogo.

  1. Primeiro, baixe o patcher atual aqui (https://www.novaragnarok.com/download/NovaPatch.zip) e extraia dentro da pasta do NovaRO.
  2. Abra o Patcher.
  3. Clique com o botão direito dentro do Patcher > use Manual Patch... e então escolha o diretório onde está localizado o patch.

Arquivos dos patches manuais:

  • Patch ID 1014 a 1144 (2017-09-29 a 2017-12-19): https://novaragnarok.com/download/patches/patch_1014_1144.tkp
  • Patch ID 1145 a 1177 (2017-12-20 a 2018-02-09): https://novaragnarok.com/download/patches/patch_1145_1177.tkp
  • Patch ID 1178 a 1214 (2018-02-12 a 2018-03-12): https://novaragnarok.com/download/patches/patch_1178_1214.tkp
  • Patch ID 1215 a 1317 (2018-03-17 a 2018-05-21): https://novaragnarok.com/download/patches/patch_1215_1317.tkp

Erro: ".NET Framework 3.5 is missing"

1º Passo

  1. Primeiro, baixe o patcher atual aqui (https://www.novaragnarok.com/download/NovaPatch.zip) e extraia dentro da pasta do NovaRO.
  2. Verifique se está funcionando.

2º Passo

  1. Baixe os seguintes arquivos:
    • https://www.microsoft.com/en-ca/download/details.aspx?id=21
    • https://www.microsoft.com/en-ca/download/details.aspx?id=17851
  2. Instale ambos - se já estiver instalado, tente reparar ou reinstalar.

O Patcher mostra apenas uma barra branca

  1. Primeiro, baixe o patcher atual aqui (https://www.novaragnarok.com/download/NovaPatch.zip) e extraia dentro da pasta do NovaRO.
  2. Verifique se está funcionando.


Problemas com o Cliente

Erro: "Cannot init d3d or grf file has problem"

  • O Ragnarök suporta resoluções de até 2048 x 1536. Isso não pode ser modificado, pois o Ragnarök utiliza DirectX 7.
  • Se certifique que a resolução de seu monitor é de até 2048 x 1536.

Crashes em certos mapas ou com Fire Rain

  • Exemplos de mapas: pay_dun01, iz_dun03, gef_dun00, beach_dun.
  • Os passos seguintes também podem resolver o problema de quando o patcher não inicia o jogo após clicar em Start Game.
  • Aviso: isso não funciona no Windows XP.
  1. Feche o NovaRO caso ainda esteje aberto.
  2. Clique com o botão direito em NovaRO.exe > Propriedades.
  3. Selecione a aba de Compatibilidade.
  4. Selecione a opção Executar esse programa em modo com Windows XP (Service Pack 3).
  5. Marque a opção Executar esse programa como administrator.
  6. Clique em Aplicar, e então em OK.
  7. Abra novamente o NovaRO. Divirta-se!


Erro: "Cannot find File: ..."

  1. Saia do jogo.
  2. Execute o NovaRO Patcher.exe e deixe o jogo atualizar.

Erro: "Ragnashield.dll is missing"

  1. Desative seu firewall e anti-vírus.
  2. Delete a pasta do NovaRO.
  3. Reinstale o NovaRO.

O cliente fecha sozinho

  1. Baixe novamente o cliente atual: https://www.novaragnarok.com/download/clients/NovaRO_201809.zip

Não consigo clicar em nada, não consigo me movimentar, porém o chat funciona.

  1. Se você não está conseguindo jogar com o novo cliente, tente usar o Legacy Client: https://www.novaragnarok.com/download/clients/NovaRO_201701.zip (extraia dentro da pasta do NovaRO)

Erro :The procedure entry point MiniDumpWriteDump could not be located in the dynamic link library dbghelp.dll"

  1. Abra a pasta do Novaro, e renomeie o arquivo "dbghelp.dll" para "dbghelp_old.dll".
    Assim, a dll carregada será a do Windows, e não que está na pasta do jogo.

Erro :The procedure entry point MiniDumpWriteDump could not be located in the dynamic link library nvspcap.dll"

  1. Abra a pasta do Novaro, e renomeie o arquivo "dbghelp.dll" para "dbghelp_old.dll".
    Assim, a dll carregada será a do Windows, e não que está na pasta do jogo.

Quando eu ataco, aparece o erro 'attempt to call a nil value'/'IsItemUsingArrow'.

1ª Solução

  1. Execute o arquivo Nova Patcher Alternative.exe, que está dentro da pasta do NovaRO, e deixe atualizar.

2ª Solução

  1. Abra o patcher, clique com o botão direito dentro dele e clique em Reset all patches.
  2. Depois clique com o botão direito dentro do patcher e clique Retry.
  3. Deixe atualizar.

3ª Solução

  1. Baixe o Instalador Completo e atualize.

Há sprites faltando no meu jogo.

1ª Solução

  1. Execute o arquivo Nova Patcher Alternative.exe, que está dentro da pasta do NovaRO, e deixe atualizar.

2ª Solução

  1. Abra o patcher, clique com o botão direito dentro dele e clique em Reset all patches.
  2. Depois clique com o botão direito dentro do patcher e clique Retry.
  3. Deixe atualizar.

3ª Solução

  1. Baixe o Instalador Completo e atualize.

Apenas "RGB Emulation" aparece como opção gráfica

  1. Essa solução serve para quem só tem "RGB Emulation" nas opções gráficas no Setup do Ragnarök.
  2. Desmarque a opção "Disable hardware-accelerated graphics", a lista agora deve mostrar outras opções.
  3. Escolha a primeira opção gráfica (Direct3D HAL).

Eu quero jogar num sistema operacional MAC/Linux

  • Suporte para Mac e Linux estão em beta.
  • Tutorial para Mac:
    • https://gist.github.com/phaicm/cd7b66b82232b1535fbd2652d6d048e3
  • Tutorial para Linux:
    • https://gist.github.com/phaicm/cc6c876c903063720617b3b3e344ffcc
  • Outras alternativas para MAC/Linux:
    1. Máquina Virtual: Instalar uma versão do Windows em um software de Máquina Virtual, como Virtual Box, VMware, Parallels, etc.
    2. Dual Booting: Instalar uma versão do Windows em outra partição/outro HD.

Meu cliente fecha sem dar nenhum erro

  1. Esse erro geralmente é causado por softwares de bancos, como o Warsaw.
  2. Parando o Warsaw:
    1. Windows > Executar > "services.msc". Desative e pare todos os serviços GAS Tecnologia.
    2. Se o problema persistir, você deverá remover o Warsaw permanentemente de seu sistema.

Erro: "MSVCP140.dll is missing"

  1. Você deve baixar e instalar/reinstalar o seguinte arquivo:
    • VC++ Redistributables 2017 (x86): https://aka.ms/vs/15/release/vc_redist.x86.exe

Erro: "api-ms-win-crt-runtime-l1-1-0.dll is missing"

  1. Você deve baixar e instalar/reinstalar o seguinte arquivo:

Crashes ao tentar dual client

  1. Esse erro ocorre por conta do UAC (User Account Control - Controle de Conta de Usuário)
  2. Se você não quiser desativar o UAC por completo, deve desabilitar ao menos o efeito sombra:
    • Windows 7 and Vista: Seguir esse guia: https://www.howtogeek.com/howto/windows-vista/make-user-account-control-uac-stop-blacking-out-the-screen-in-windows-vista/
    • Windows 10: Clicar em "Alterar quando essas notificações aparecerem" e então em:
    • Procure pela opção de remover o escurecimento/sombreamento.


Problemas com WMI

Há varios problemas que podem ocorrer relacionados ao WMI (Windows Management Instrumentation - Instrumentação de Gerenciamento do Windows).

Usando Windows XP (RagnaShield Beta - "WMI generic failure")

  1. Windows > Executar
  2. Digite services.msc, e então OK.
  3. Procure por Instrumentação de Gerenciamento do Windows.
  4. Inicie o serviço.

Error: "0x80070422"

  1. Windows > Executar
  2. Digite services.msc, e então OK.
  3. Procure por Instrumentação de Gerenciamento do Windows.
  4. Clique com o botão direito em > Propriedades > Mude o Tipo de inicialização para Automático.
  5. Clique em Iniciar, e então em OK.

Erros: "0x80041002 (oi 0x80041013, 0x80041010)"

1ª Solução

  1. Executar o arquivo wmi_rebuild.bat na pasta do NovaRO e clique com o botão direito em > Executar como administrador.
  2. Após finalizar, reinicie seu computador.
  3. Se você tiver erros ao executar, tente a 2ª solução.

Solution 2

  1. Windows > Executar
  2. Digite services.msc, e então OK.
  3. Procure por Instrumentação de Gerenciamento do Windows.
  4. Clique com o botão direito em > Propriedades > Mude o Tipo de inicialização para Desativado.
  5. Reinicie seu computador.
  6. Vá até a pasta C:\Windows\System32\wbem\
  7. Renomeie repository para repository.old.
  8. Abra novamente os Serviços (passos 1 a 3).
  9. Clique com o botão direito em > Propriedades > Mude o Tipo de inicialização para Desativado.
  10. Clique em Iniciar, e então em OK.


Atualização - Novo Cliente 2018

Erro: "Win32 application error"

  1. Esse erro ocorre devido a um bug no patcher padrão.
  2. Você pode baixar os arquivos manualmente em https://www.novaragnarok.com/download/clients/NovaRO_201809.zip
  3. Se isso não ajudar, use o cliente Legacy 2017 (https://www.novaragnarok.com/download/clients/NovaRO_201701.zip).
  4. Você pode tentar também a versão atualizada do Patcher (https://www.novaragnarok.com/download/NovaPatch.zip) sendo que o Patcher baixará o cliente sem problemas.

"Este aplicativo não pode ser executado em seu PC" (Windows 10)

  1. Você pode baixar os arquivos manualmente em https://www.novaragnarok.com/download/clients/NovaRO_201809.zip
  2. Se isso não ajudar, use o cliente Legacy 2017 (https://www.novaragnarok.com/download/clients/NovaRO_201701.zip).


A minha tela está assim, o que eu faço?

  1. Atualize ou reinstale o DirectX e seus drivers gráficos.
  2. Tenha certeza de escolher a opção gráfica certa no Setup.
  3. Se isso não ajudar, use o cliente Legacy 2017 (https://www.novaragnarok.com/download/clients/NovaRO_201701.zip).


Problemas dentro do jogo

Screenshot: O botão PrintScreen não funciona

  1. Esc > Options and Settings > Interface.
  2. Mude o atalho da ação "Screenshot".

Screenshot: A imagem está cinza e/ou distorcida

  1. Entre na pasta do NovaRO e abra o arquivo Setup.exe.
  2. Mude sua resolução horizontal para que ela seja divisível por 4.
  3. (Se você está usando a opção de falsa tela cheia (false fullscreen), terá de mudar também em NovaRO > RagnaShield > config.ini.)

Os atalhos não estão salvando

  1. Baixe o cliente atual e extraia na pasta do NovaRO.
  2. Se isso não resolver, baixe o cliente Legacy 2017 e extraia na pasta do NovaRO.

Erro: "Failed to Connect to Server"

  1. Certifique-se que o servidor não está em manutenção/fora do ar (você pode verificar no Discord)
  2. Tente reinstalar o jogo em outra pasta ou em outro disco local.
  3. Abras as seguintes portas: 6954, 6175 and 5175.
  4. Contate um GM no fórum ou no Discord.

A fonte do jogo está muito pequena

  1. Vá até o arquivo NovaRO\RagnaShield\import_config.ini e mude os valores dentro dele:
  2. Remova // da linha //Font.Size: 15.
  3. Faça o mesmo para a linha Font.Family (aqui você pode escolher qualquer fonte do seu sistema).
  4. As fontes recomendadas são:
    • Verdana 12 para não confundir I e i.
    • Arial 13 para fontes maiores.

Não consigo atribuir pontos de skill após um skill reset

Você deve desequipar todos os items que dão acesso à alguma skill antes de resetar. Alguns itens que afetam isso são:

  •  Heroic Backpack
  •  Silverwork Bracelet

Para ter certeza, remova todos seus equipamentos antes de resetar.


Problemas com o NovaExt

  • Feche todos os clientes abertos antes de editar o NovaExt.ini.﻿
  • Vá até NovaRO\Setup.exe, e desmarque a opção "Restrict mouse to window".
  • Mude o "Sound Mode" para "Use 2D Sound".
  • Instale todos as dependências VC++.
  • Verifique se seu cliente tem o nome NovaRO.exe (qualquer outro nome não funcionará).
  • Tenha certeza que você não está executando o cliente em modo de compatibilidade, e de que você está executando ele como um administrador.
  • Se você ainda tiver problemas, poste seu __NovaExt.log e NovaExt.ini (estão dentro da pasta do NovaRO) e as especificações de seu computador (Intel/AMD, GPU, etc), Sistema Operacional, e qual anti-vírus você usa no seguinte tópico:
    • https://www.novaragnarok.com/forum/topic/7543-novaext-troubleshooting-problems/﻿
  • (Se você não souber inglês, poste em Português que um GM ou player traduzirá).


Problemas relacionados a GMs

Joguei um item no chão sem querer / refinei o equipamento errado / resetei os encantamentos sem querer / vendi um item sem querer no NPC, o GM pode recuperar?

Não. Os GMs não irão devolver items nesses tipos de casos que acontecem por conta de falta de atenção de players.

Coloquei o item pra venda com um 0 a menos, e vendi sem ver - os GMs podem me devolver?

Não. Os GMs não podem pedir para o jogador que comprou para devolver o seu item, já que não seria justo com ele.

Achei um bug, o que eu faço?

Poste no fórum, na seção "Bug Tracker", com o máximo de detalhes possíveis (screenshots, descrição, gravações).

  • Novamente, se o inglês não for o seu forte, um GM ou player irá te ajudar com a tradução.

Tem um player se comportando mal, o que eu faço?

Antes de tudo, veja se as regras do servidor foram infringidas antes de reportar alguém.

Se possível, faça também um report no fórum, na seção "Player Reports", com o máximo de detalhes possíveis (screenshots, descrição, gravações).
