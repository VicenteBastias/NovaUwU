Skill is used to make a plain link to the Divine Pride skill page.

Syntax

{{Skill|id= Name of the Skill}}

Parameter
id
The unique ID of the skill.
Name of the Skill
The name of the skill, this is not a named parameter so you don't need to use the = sign.

You can find the id of every skill at Skill Links

Example

Using {{Skill|id=1 Basic Skill}} will result into: Basic Skill
