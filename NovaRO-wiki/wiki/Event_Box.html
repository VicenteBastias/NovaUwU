Event Tickets are a special kind of currency exclusive to NovaRO. They can be exchanged for either Event Eggs or  Gold Coins.

Event Boxes

Every Event Box contains the same list of consumables and a shared set of hats depending on their series, but also a separate set of exclusive costumes/pets that are themed and unique to that specific box, so choose wisely! New boxes are added periodically.


Event Boxes can be purchased for 10  Event Tickets each for Series 1 Eggs or 15  Event Tickets each for Series 2 Eggs , from the Event Ticket Exchanger, located on the Third Floor of the Main Office. Series 1 and 2 eggs have a different set of consumables and a different set of static hats.


The Ticket Exchanger will also trade  Event Tickets to  Gold Coins, on a 1:1 ratio, if you don't want Event Boxes.
