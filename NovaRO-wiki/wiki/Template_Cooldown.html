Template:Cooldown

  • Language
  • Watch
  • Edit

Cooldown is used to add a table that has information about the cooldown of quests and instances.

Syntax:

{{Cooldown|time=Time to reset}}

You can use up to three parameters with this template.

Parameters
time
Shows the cooldown time before the player can repeat the task.
bg
(Optional) Changes the title background color, if it's not specified the default will be used.
col
(Optional) Changes the title text color, if it's not specified the default will be used.
Both bg and col accept colors value according to the W3C CSS Colors specification.

Examples:

Using {{Cooldown|time=3 Hours}} will result into:

This quest or instance is repeatable.
The cooldown is 3 Hours.

Using {{Cooldown|time=3 Hours|bg=#76e876}} will result into:

This quest or instance is repeatable.
The cooldown is 3 Hours.

Using {{Cooldown|time=3 Hours|bg=#76e876|col=#d808d2}} will result into:

This quest or instance is repeatable.
The cooldown is 3 Hours.
