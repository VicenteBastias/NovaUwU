Essential Information

Rates

  • Base EXP Rate: 25x
  • Job EXP Rate: 25x
  • Quest EXP Rate: 25x
  • Item Drop Rate: 10x
  • MVP Drop Rate: 10x
  • MVP Card Drop Rate: 0.02%

Other Information

  • Max Base Level: 250
  • Max Job Levels: 50/70/70/50 (1st/2nd(Trans)/3rd/4th job)
  • Max Stats: 130 (3rd job)
  • Max Traits: 100 (4th job)
  • Max ASPD: 193
  • Party Share Limit: 30 Base Levels

Official Server Features

  • Episode: 17.2
  • Third Classes: Fully Available
  • Rebellion/Kagerou/Oboro/Doram: Fully Available
  • Functional Eclage, El Dicastes, Malangdo & Mora Quests
  • Instance Dungeons Available: Central Laboratory, Last Room, Charleston Crisis, Airship Assault, Bios Island, Morse's Cave, Temple of the Demon God, Faceworm's Nest, Old Glast Heim, Endless Tower, and more!
    See the Instances page for more information.
  • Illusion Dungeons
  • Nightmare Mode Dungeons
    Nightmare Clock Tower, Glast Heim, Pyramids, Scaraba Hole
  • Champion Mob System
  • WoE, PvP, and Battlegrounds
