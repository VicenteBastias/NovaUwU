GuideBox is used to give basic information about the class the guide refers, the author name and the job stats bonus.

Syntax

{{GuideBox|job=|image=|basejob=|author=|str=|agi=|vit=|int=|dex=|luk=|bgColor=}}

Parameters
job
The job that the guide is about.
image
The name of image file that was uploaded to the wiki, if the file extension is not present it will default to PNG.
basejob
The name of the first job of the class.
author
The author name.
str
The STR job bonus of the class.
agi
The AGI job bonus of the class.
vit
The VIT job bonus of the class.
int
The INT job bonus of the class.
dex
The DEX job bonus of the class.
luk
The LUK job bonus of the class.
To omit the job bonuses table just don't use any of the stats.
bgColor
(Optional) Changes the title background color, if it's not specified the default will be used.
bgColor accepts colors value according to the W3C CSS Colors specification.

Example

Using {{GuideBox|job=Guillotine Cross|image=Guillotinecross|basejob=Thief|author=WikiAuthor|str=8|agi=16|vit=6|int=5|dex=9|luk=4}} will result into:

Guillotine Cross
Job Base: Thief
Written By: WikiAuthor
Job Bonuses
STR AGI VIT INT DEX LUK
8 16 6 5 9 4


Using {{GuideBox|job=Guillotine Cross|image=Guillotinecross|basejob=Thief|author=WikiAuthor}} will result into:

Guillotine Cross
Job Base: Thief
Written By: WikiAuthor
Job Bonuses
STR AGI VIT INT DEX LUK
