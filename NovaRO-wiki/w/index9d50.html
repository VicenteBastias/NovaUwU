Skill info is used to create a link directed at the skill in the class skill template page. The link is composed of an icon and the name of the skill.

Syntax

{{Skill Info|class=|id=|skill=}}

Parameters
class
The name of the class that the skill belongs to.
id
The unique ID of the skill.
skill
The name of the skill.

Class names available to be used with this template can be consulted here.

Alternatively you can check the following page with all the links ready to be used - Skill Links.

Example

Using {{Skill Info|class=Novice|id=1|skill=Basic Skill}} will result into: Basic Skill.

Using {{Skill Info|class=Rune Knight|id=2007|skill=Dragon Training}} will result into: Dragon Training.
