Line of skills to be used in Template:Skill Tree.

Syntax

{{Skill_Tree_Line|bottom=|short names=|class=|id1=|skill name1=|id2=|skill name2=|id3=|skill name3=|id4=|skill name4=|id5=|skill name5=|id6=|skill name6=|id7=|skill name7=}}

Parameters
bottom
(Optional) If you want names below skill icons instead of the default option at the top
yes if you want it on
short names
(Optional) If you want shorter names similar to how it's displayed in game
on if you want it on
class
class of the template used (don't forget to use _ when needed instead of a space)
id
the id of the skill to show
skill name
name of the skill to display
skill level
(Optional) to show the level of the skill
max level
(Optional) to show the max level of the skill

Supports up to 7 id and 7 skill names of the same template. Check Template:Skill Tree for more info.

Not all slots need to be used, any can be left empty.

Examples

Example #1
Basic Skill Tree Line with skills and their names
{{Skill_Tree_Line
|class=Acolyte
|id1=33
|skill name1=Angelus
|id2=31
|skill name2=Aqua Benedicta
|id3=34
|skill name3=Blessing
|id4=35
|skill name4=Cure
|id5=30
|skill name5=Decrease Agility
|id6=23
|skill name6=Demon Bane
|id7=22
|skill name7=Divine Protection
}}
Angelus Aqua Benedicta Blessing Cure Decrease Agility Demon Bane Divine Protection
