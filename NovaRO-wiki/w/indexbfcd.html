Skill List is used to create a link directed at the skill page in Divine Pride. The link is composed of an icon and the name of the skill.

It's really similar to Template:Skill Info, the difference only being the destination.

Syntax

{{Skill List|id=|skill=}}

Parameters
id
The unique ID of the skill.
skill
The name of the skill.

Example

Using {{Skill List|id=28|skill=Heal}} will result into: Heal.
