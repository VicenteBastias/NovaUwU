Skill Tree is used to create skill trees similar to the ones in game from a single class.

Syntax

Parameters
title
name that appears at the top of the table related to a Skill Tree
note: if you make use of the custom name below, you don't even need to add a title
example: Super Novice will make the table display Super Novice Skill Tree
custom name (optional)
in case you don't want to use something related to a Skill Tree, you can add a different name here
example: Strong Bolter, I got the Power! will display that same exact text
skills
this is where you use the Template:Skill Tree Line to add lines with skills
each skill needs to have an id and a skill name
you don't need to fill the seven slots, only fill the ones you need/want
is advised to check the examples to get an idea of how it works

Usage

  • each skill to be displayed needs one id and one skill name that ranges from 1 to 7
  • you need to use the Template:Skill Tree Line for this to work properly for each line of skills
  • each usage of the Template:Skill Tree Line makes use of a different template. this means that you can customize tables as you want as long as it's on the same line of skills
  • this template supports how many lines full of skills as you can add and want displayed. it's flexible!!

Examples

Example #1 - Single Line, default class title
{{Skill Tree
|title=Acolyte
|skills=
{{Skill_Tree_Line
|class=Acolyte
|id1=33
|skill name1=Angelus
|id2=31
|skill name2=Aqua Benedicta
|id3=34
|skill name3=Blessing
|id4=35
|skill name4=Cure
|id5=30
|skill name5=Decrease Agility
|id6=23
|skill name6=Demon Bane
|id7=22
|skill name7=Divine Protection
}}}}
Acolyte Skill Tree
Angelus Aqua Benedicta Blessing Cure Decrease Agility Demon Bane Divine Protection
Example #2 - Single Line, custom title, but only displaying a few skills
{{Skill Tree
|custom name=Buffer, the Monster Slayer
|skills=
{{Skill_Tree_Line
|class=Acolyte
|id3=34
|skill name3=Blessing
|id7=29
|skill name7=Increase Agility
}}}}
Buffer, the Monster Slayer
Blessing Increase Agility
Example #3 - Multi-lines, custom title, skills from more than a single class
{{Skill Tree
|custom name=Pope 2.0
|skills=
{{Skill_Tree_Line

|skill name7=Divine Protection
}}
{{Skill_Tree_Line
|class=Priest
|id2=68
|skill name2=Aspersio
|id5=75
|skill name5=Gloria
}}
{{Skill_Tree_Line
|class=Arch_Bishop

}}}}
