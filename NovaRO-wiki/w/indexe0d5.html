Used for linking to card with an automatic unclickable thumbnail. The card link is now targeted to NovaRO Market.

Syntax =

{{Card|id= Name of the Card}}

Parameters
id
The unique ID from the card.
Name of The Card
Replace this with the actual name of the card, there should be an empty space between the id and the name.

Example

Using {{Card|id=4431 Kasa Card}} will result into: Kasa card.
